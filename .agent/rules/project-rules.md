# PROJECT RULES

## Product Summary

A marketplace that allows sharing unused spaces by day or time slot.
All negotiations, meetings, and payments happen OFF-PLATFORM.

## User Roles

- Guest:
  - Search listings
  - View listing details
- User:
  - Create listings
  - Edit own listings
  - Favorite listings
  - Unlock contact information
- Admin:
  - Approve or disable listings
  - Manage users and coin balances

## Monetization Rules

- Contact unlock is paid per listing using coins
- Optional subscription allows unlimited unlocks
- Coins are topped up via SePay
- Coin rewards may be granted when listings are approved

## Listing Lifecycle

- Listing creation is free
- Listing MUST be approved before public visibility
- Editing an approved listing resets status to `pending`
- Listings are NEVER deleted, only disabled
