# LANDING PAGE

This page must reflect the core philosophy:

- Connection-only marketplace
- No booking
- No reservation handling
- No user-to-user payment processing

### 2. How It Works (3 Steps)

1. Tìm kiếm theo khu vực
2. Xem chi tiết & mở khóa thông tin liên hệ
3. Liên hệ trực tiếp để trao đổi

Important:
The flow MUST NOT imply booking confirmation,
availability guarantee, or platform-managed scheduling.
